Summary of WinLIKE form submit samples

Normal submits within windows (#42):
    <form id=form1>
       <input name=eingabe type=text>
       <input type=submit name=widget value=button>
       <a href="javascript:document.getElementById('form1').submit();">Submit per Script</a>
    </form>

Submits within windows with WinLIKE help (#42):
    <script src="...../winlike/winman/winxtra.js"></script>
    <form id=form1 onSubmit="return submitWinLIKE_(event)">
       <input name=eingabe type=text>
       <input type=submit name=widget value=button>
    </form>

Submits in foreign windows (#43/#44):
    <script src="...../winlike/winman/winxtra.js"></script>
    <form id=form1 target=new_win onSubmit="return submitWinLIKE_(event)">
       <input name=eingabe type=text>
       <input type=submit name=widget value=button onClick="submitWinLIKE_(event)">
    </form>

Submits in foreign windows with ASP.NET System.Web.UI.HTMLControls (#43):
     <script src="...../winlike/winman/winxtra.js"></script>
     <form runat=server id=Form1 target=new_win onsubmit="return submitWinLIKE_(event)">
        <input runat=server id=text type=text>
        <input type=submit name=widget value=button onClick="submitWinLIKE_(event)"
                                                              runat=server onserverclick=myserverhandler>
     </form>

Submits in foreign windows with ASP.NET System.Web.UI.WebControls (#43):
    ASPX file:
       <script src="...../winlike/winman/winxtra.js"></script>
       <form runat=server id=Form1 target=new_win onsubmit="return submitWinLIKE_(event)">
          <input runat=server id=text type=text>
          <asp:button runat=server id=Button1 text=button onClick=myServerHandler></asp:button>
       </form>
    Add to Page_Load function:
       Button1.Attributes.Add("onClick", "submitWinLIKE_(event)");

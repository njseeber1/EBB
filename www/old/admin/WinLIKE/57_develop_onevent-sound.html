This page demonstrates the onEvent-property of a window. The called event-functions are defined in the root-page and can abort some actions (close and resize here)!
For instance, you can use this feature to track user behaviour by requesting an URL with data in a hidden IFRAME at each user action.
For other events like 'onFocus' or 'lostFocus' you can use the standard browser events.

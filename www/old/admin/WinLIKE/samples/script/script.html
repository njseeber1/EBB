You can use the function WinLIKE.searchwindow(windowName) to look for a window by its name. You can subsequently change some attributes of the window (e.g. Ski, Cls, ...) manually and later use the function win.draw() to render the WinLIKE-window again:

var winNr=parent.WinLIKE.searchwindow("scriptWindow");
parent.WinLIKE.windows[winNr].Cls=false;
parent.WinLIKE.windows[winNr].Ski='red';
parent.WinLIKE.windows[winNr].Mov=false;
parent.WinLIKE.windows[winNr].draw();
